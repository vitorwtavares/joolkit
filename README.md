`joolkit` is built for people who still want to write their own applications, but are tired of retyping the same details, juggling files, and keeping status updates in scattered tabs.

It brings the repetitive parts of applying into one place: quick personal info copy, cover letter variations with token replacement, reusable answers, and a lightweight application tracker.

## Features

- **Quick Copy** for personal details, links, and frequently used files
- **Cover Letter Editor** with editable templates, variation switching, token substitution, and PDF export
- **Answer Bank** for saving reusable short and long-form responses
- **Application Tracker** for managing roles, statuses, notes, deadlines, and follow-ups

## Why it exists

- Avoid manual busywork without handing your applications over to AI tools that spray the same application everywhere
- You stay in control of the writing
- Repetitive admin work gets faster
- The workflow stays focused and desktop-friendly

## Stack

- **Frontend:** React, Vite, Tailwind CSS, shadcn/ui, Tiptap, TanStack Query
- **Backend:** Node.js, Express, Supabase, Puppeteer Core
- **Infra:** Vercel, Supabase
